2 #permalink Forms

Normal <input> with <label> wrapper

Example 
<label for="input-text">Normal Input</label>
<input name="input-text" type="text">

2.1 #permalink Required

Adds asterisks after label. Creating a span with the words "Required" for accessibility purposes.

Example 
<label for="input-required">Required Input <span class="span--required">(required)</span></label>
<input required name="input-required" type="text">

2.2 #permalink Inline

If you want the label on the left side of the <input>

Example 
<label class="label--inline" for="input-inline">
  Inline Input
  <input name="input-inline" type="text">
</label>

2.3 #permalink Radio

add to <label> if <input type="radio">

Example 
<label class="label--radio" for="input-radio-1">
  <input name="radio-group" id="input-radio-1" type="radio">
  Inline Input
</label>
<label class="label--radio" for="input-radio-2">
  <input name="radio-group" id="input-radio-2" type="radio">
  Inline Input
</label>

2.4 #permalink Checkbox

Example 
<label class="label--checkbox" for="input-checkbox">
  <input id="input-checkbox" type="checkbox">
  Checkbox
</label>

2.5 #permalink Submit Buttons

Example 
<input type="submit" value="Submit Input">
<button>Submit Button</button>
