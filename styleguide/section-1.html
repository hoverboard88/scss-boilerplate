1 #permalink Variables

SCSS Variables

1.1 #permalink Grid

Declare variables for grid.

  • $grid__columns: (integer) Amount of columns in grid.
  • $grid__mobile-breakpoint: (width) Breakpoint for when "mobile" becomes "desktop".

1.2 #permalink Booleans

Declare variables for grid.

  • $border-box: (Boolean) Turn box-sizing: border-box on or off.

1.3 #permalink Colors

To use colors throughout stylesheets

1.4 #permalink Base Styles

Base variables

  • $border-color: (color) Used with font color and <hr>'s.
  • $border-border: (color) Used with input borders.
  • $base--font-family: (font stack) Base body font.
  • $secondary--font-family: (font stack) Display font font.

1.5 #permalink Font Weights

set "bold" and "normal" weights here and use @include font-weight() mixin instead of font-weight.

  • $font-weight__normal: (font-weight declaration) Used in font-weight() mixin.
  • $font-weight__bold: (font-weight declaration) Used in font-weight() mixin.

1.6 #permalink Vertical Rhythm

Set base font-size and line-height for vertical rhythm

  • $base--font-size: (size in pixels) Used for vertical rhythm grid.
  • $base--line-height: (height in non-units) Used for vertical rhythm grid.

1.7 #permalink Headings

Set font-size for Headings

  • $font-size--h1: (size in pixels) <h1> size.
  • $font-size--h2: (size in pixels) <h2> size.
  • $font-size--h3: (size in pixels) <h3> size.
  • $font-size--h4: (size in pixels) <h4> size.
  • $font-size--h5: (size in pixels) <h5> size.
  • $font-size--h6: (size in pixels) <h6> size.

1.8 #permalink Dropdowns

Values for dropdowns

  • $dropdown__width: (size in pixels) Width of dropdowns.
  • $dropdown__width--offset: (size in pixels) If your boxes have borders or padding.

1.8 #permalink Other Sizes

Set font-size for others

  • $font-size--hero: (size in pixels) Hero size.
  • $font-size--small: (size in pixels) <small> size.
